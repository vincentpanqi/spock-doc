## 重启服务组
 
* **URL:**

    `/stacks/:stackName/restart`

* **Method:**

    `POST`

* **URL Params:**

    `None`

* **Data Params:**

    `None`

* **Success Response:**

	**Code:** `200`

* **Error Response:**

	**Code:** `624`
  	
  	**Content:** `{ "message" : "Restart Stack Error", "code": "624"}`
