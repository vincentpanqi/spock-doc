---
title: 停止服务接口
taxonomy:
    category: docs
---

* **URL:**

    `/stacks/:stackName/services/:serviceName/stop`

* **Method:**

    `POST`

* **URL Params:**

    `None`

* **Data Params:**

    `None`

* **返回:**

	**Code:** `200`

* **错误信息:**

	**Code:** `633`
  	
  	**Content:** `{ "message" : "Stop Service Error", "code": "633"}`
