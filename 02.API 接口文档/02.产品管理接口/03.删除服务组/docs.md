## 删除服务组
 
* **URL:**

    `/api/stacks/:stackName`

* **Method:**

    `DELETE`

* **URL Params:**

    `None`

* **Data Params:**

    `None`

* **Success Response:**

	**Code:** `200`

* **Error Response:**

	**Code:** `623`
  	
  	**Content:** `{ "message" : "Delete Stack Error", "code": "623"}`
