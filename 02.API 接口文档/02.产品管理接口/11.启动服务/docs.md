    category: docs
---

* **URL:**

    `/stacks/:stackName/services/:serviceName/start`

* **Method:**

    `POST`

* **URL Params:**

    `None`

* **Data Params:**

    `None`

* **Success Response:**

	**Code:** `200`

* **Error Response:**

	**Code:** `634`
  	
  	**Content:** `{ "message" : "Start Service Error", "code": "634"}`
