---
title: 重启服务接口
taxonomy:
    category: docs
---

* **URL:**

    `/stacks/:stackName/services/:serviceName/restart`

* **Method:**

    `POST`

* **URL Params:**

    `None`

* **Data Params:**

    `None`

* **返回:**

	**Code:** `200`

* **错误信息:**

	**Code:** `632`
  	
  	**Content:** `{ "message" : "Restart Service Error", "code": "632"}`
