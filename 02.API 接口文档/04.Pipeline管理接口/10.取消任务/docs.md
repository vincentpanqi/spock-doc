* **URL:**

    `/api/pipeline/tasks/:taskID/cancel`

* **Method:**

    `POST`

* **URL Params:**

	`None`

* **Data Params:**

    `None`

* **返回:**

	**Code:** `200`

* **错误信息:**

	**Code:** `688`
  	
  	**Content:** `{ "message" : "Cancel Task Error", "code": "688", "description": "error details"}`
