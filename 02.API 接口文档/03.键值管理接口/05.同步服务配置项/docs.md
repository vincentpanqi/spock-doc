* **URL:**

    `/api/configs/sync`

* **Method:**

    `POST`

* **URL Params:**

    **Optional:** 默认不填是false

    `force=<false|true>`

* **Data Params:**

    `None`

* **返回:**

    **Code:** `200`

* **错误信息:**

    **Code:** `625`
  	
  	**Content:** `{ "message" : "Sync Config Service Specs Error", "code": "625"}`
